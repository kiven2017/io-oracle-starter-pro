# 11. Token Utility

## 11.1 Token Positioning
AIOracle Token ($AIO) is the native utility token of the AIOracle network. It supports trusted data infrastructure, AI oracle services, node incentives, governance participation, and ecosystem expansion.

## 11.2 Core Utilities
**Node Staking**: $AIO serves as the primary staking asset for critical node roles.
**Node Rewards and Settlement**: It rewards nodes providing AI inference, data cleaning, validation, and execution.
**Payment for AI Oracle Services**: It functions as a settlement asset for trusted data requests, risk scoring, and AI result delivery.
**Governance Participation**: Holders may participate in parameter updates, treasury decisions, and protocol upgrades.
**Ecosystem Coordination**: It may be used for grants, partner incentives, and compute subsidies.

## 11.3 Sources of Demand
Demand is expected to come from node staking, service payments, governance participation, and ecosystem coordination.

## 11.4 Long-Term Value Logic
The long-term value of $AIO should be grounded in network service value rather than market narrative alone. More devices, more trusted data, more AI requests, and more service revenue strengthen the token’s network role.
