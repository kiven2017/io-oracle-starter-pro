# 11. Token Utility｜代幣效用

## 11.1 代幣定位
AIOracle Token（$AIO）是 AIOracle 網路中的原生功能型代幣，用於支撐可信資料網路、AI 預言機服務、節點激勵、治理參與與生態擴張。AIOracle 的定位並非單純搬運鏈下資料，而是生成可被智能合約、RWA 協議與企業系統調用的「鏈上可用的物理事實」。

## 11.2 核心效用
**節點質押**：$AIO 是關鍵節點的主要質押資產。
**節點獎勵與服務結算**：用於向提供 AI 推理、資料清洗、驗證與執行的節點分發獎勵。
**AI 預言機服務支付**：作為可信資料請求、風險評分與 AI 結果交付的結算媒介。
**治理參與**：持幣者可參與節點參數、排放調整、國庫使用與協議升級治理。
**生態協同**：可作為開發者補助、合作夥伴激勵與算力補貼工具。

## 11.3 代幣需求來源
代幣需求主要來自節點質押、服務支付、治理需求與生態合作需求。

## 11.4 長期價值邏輯
AIOracle 的代幣價值應建立在網路服務價值之上，而非僅依賴市場敘事。更多設備接入、更多可信資料、更多 AI 請求與更多服務費，將共同強化 $AIO 的網路價值。
