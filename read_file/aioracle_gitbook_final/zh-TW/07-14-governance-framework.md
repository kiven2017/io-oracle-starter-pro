# 14. Governance Framework｜治理框架

## 14.1 治理定位
AIOracle 採取基金會協調 + 協議治理 + 社群參與的多層治理模式，以兼顧安全、商業推進、技術演進與長期可持續性。

## 14.2 治理主體
- 可信網路基金會
- 香港财富通控股集团有限公司（運營實體）
- 節點參與者
- $AIO 持幣者與社群

## 14.3 治理原則
安全優先、漸進去中心化、聲譽與貢獻導向、長期價值對齊。

## 14.4 治理範圍
涵蓋協議參數治理、生態資源配置、技術升級與治理結構演進。

## 14.5 提案與投票
依風險級別分為一般提案、重要提案與高風險技術提案。

## 14.6 緊急治理
在合約漏洞、跨鏈事件或虛假資料大規模提交等極端情況下，可啟動有限緊急治理機制。

## 14.7 國庫治理
國庫優先服務網路增長、基礎設施安全與高價值場景接入。

## 14.8 演進路徑
治理將從基金會主導逐步演進到更高程度的鏈上治理、委託投票與聲譽加權參與。
