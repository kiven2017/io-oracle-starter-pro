# 21. Token Release and Vesting Appendix｜代幣釋放附錄

正式融資版白皮書需要將分配比例與釋放節奏拆解為更容易被審閱的結構。對投資人而言，最重要的不只是比例本身，而是哪些份額在 TGE 即具備流動性、哪些份額受 Cliff 約束、哪些份額會在中期形成主要潛在供給壓力。

AIOracle 採取的設計原則，是讓節點與團隊在 TGE 時保持極低即時流通壓力，而把更高的早期市場功能交給流動性池與有限比例的預售釋放。這有助於平衡價格發現、節點長期激勵與市場信心。
